Project Title

Video Game Sales Analysis | SQL + Power BI

Overview

This project analyzes global video game sales data using SQL and Power BI. The objective is to identify sales trends, top-performing games, console performance, publisher contributions, and regional sales patterns.

Key Insights
Action games generated the highest global sales.
Rockstar Games published several top-selling titles.
PS4 and Xbox 360 contributed significantly to total sales.
Higher critic scores generally correlated with stronger sales performance.
